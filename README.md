# tjappen

Welcome to **tjappen**, a minimalist digital recipe directory built with Hugo and Tailwind CSS.

## 📖 Developer & Contributor Documentation

All project documentation, design guidelines, and contribution instructions are organized in the `docs/` folder:

- **Technical Specification (spec.md)** - Details about the core stack, page bundle structures, image pipeline, client-side search, and UX behavior.
- **Design Style Guide (style-guide.md)** - Principles of the Warm Minimalist design identity, color schemes, typography, and interactive aesthetics.
- **Photography Standard (photo-guide.md)** - Guidelines for photography style, tone, lighting, framing, composition, and styling.
- **How to Add a Recipe (add-recipe.md)** - Step-by-step guide on creating page bundles, naming assets, and writing/scaling ingredient front-matter.
